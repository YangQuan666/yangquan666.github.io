## Welcome to GitHub Pages

## Build Setup

```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run docs:dev

# build for production and launch server
$ yarn run docs:build
$ yarn run docs:serve
```
